How to create HTML file?

  • As we all known that, HTML file is a text file, so to create an HTML file we can use any text editors.
  • There are professional text editors available which you can download, but for learning HTML we recommend the best text editor is Notepad (Windows) or TextEdit (Mac) or RK Editor (Online Editor).
  • After learning the basics, you can easily use other professional text editors.
  • Follow the steps below to create your first web page with Notepad or TextEdit or RK Editor.

1. How to open Notepad (Windows) and create HTML file?

A. Click on Start or the Windows orb logo on the taskbar at the bottom of the desktop to show the Start menu.

B. Click on Programs or All Programs to show a menu of installed programs in Windows.

C. Click on the Accessories folder to expand it.

D. Click on the Notepad icon to open the program.

E. On opening the notepad you can write some HTML document in notepad.

F. Then click on the file button at the left top. After that click on the save button to open the file saveing option.

G. Name the file with .html or .htm extension, for example rk.html or rk.htm. And then save the file by clicking save button.

NOTE : It is important that the extension .html or .htm is specified some text editors, such as Notepad, will automatically save it as .txt .

2. How to open Textedit (Mac) and create HTML file?

A. Click on launcher button on the taskbar at the bottom of the desktop to show the search bar and many more.

B. Search textedit in the search bar which is at the top of the screen. and click on the textedit icon in the search results.

C. now after opening textedit click on the new document to create text file in which we can write our first program.

D. write some HTML document in textedit.

E. Then click on the file button at the left top to open file saving options.

F. Click on the save button. Name the file with .html or .htm extension, for example rk.html (or select the file format as html). And then save the file by clicking save button.

NOTE : It is important that the extension .html or .htm is specified some text editors, such as textedit, will automatically save it as .txt .

3. How to view file in browser?

To open the saved HTML file in your browser. First go to the file where you have saved it. Then double click on the file, or right-click and choose Open with to open it in your favorite browser.
