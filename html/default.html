What is HTML?

  • HTML is the most simple and text oriented programing language,
  • HTML stands for HyperText Markup Language. which is used to creat world wilde web documents.
  • HyperText is ordinary text that has been dressed up with extra feature such as formating, images, multimedia and links to other documents. A text has a link within it, is a hypertext. HyperText is a way to link two or more web pages (HTML documents) with each other. So HTML is hyperlink specification language
  • Markup is the process of taking ordinary text and adding extra symbols (tags). Each symbol used for markup in HTML is a command that tell the browser how to display the text.
  • Using this language user can create web page which can be viewed in any web browser. As all the text on this page you are reading right now lives inside HTML tags
  • Web Page : A web page is a document which is commonly written in HTML and translated by a web browser. A web page can be identified by entering an URL. A Web page can be of the static or dynamic type. With the help of HTML only, we can create static web pages
  • HTMl is essentially a set of instructions to web browser for formatting and layout of web page
  • HTML provides structure to the content appearing on a website, such as images, text, or videos.
  • HTML is not a programming language. It is a standard markup language in which most websites are written.

Applications of HTML

  • Web pages development : HTML is used to create pages which are rendered over the web. Almost every page of web is having html tags in it to render its details in browser.
  • Internet Navigation : HTML provides tags which are used to navigate from one page to another and is heavily used in internet navigation.
  • Responsive UI : HTML pages now-a-days works well on all platform, mobile, tabs, desktop or laptops owing to responsive design strategy.
  • Game development : HTML5 has native support for rich experience and is now useful in gaming developent arena as well.
  • Offline support : HTML pages once loaded can be made available offline on the machine without any need of internet.

Features of HTML

  • HTML is a very easy and simple language. It can be easily understood and modified. HTML is relatively easy to learn because every tag is predefined, so only we need to know the work of tags and its attributes.
  • Document prepared in HTML include reference graphics and formating tags.
  • HTML contains powerful formatting facilities for text, page, images, etc. And alos It facilitates the programmer to add Graphics, Videos, and Sound to the web pages which makes it more attractive and interactive
  • HTML is very easy to make an effective presentation with HTML because it has a lot of formatting tags.
  • Html is a platform Independent because it can be displayed on any platform like Windows, Linux, and Macintosh, etc.
  • HTML is a case-insensitive language, which means we can use tags either in lower-case or upper-case.
  • HTML supports a wide range of colors, formats, and layouts.
  • HTML support forms which make it possible to create documents that collect and process user input

  • HTML is Easy to Learn and Use.

  • HTML will not cost any thing for it's use. There are no expensive licenses to buy.

  • HTML can Integrate Easily with Other Languages.

  • Html is the most search engine friendly.

Disadvantages of HTML

  • HTML is not a programming language.
  • any simple calculation can not be done in HTML.
  • Html is used to create only plain or static pages. If one wants dynamic pages then HTML is not useful. Interactiveweb pages cannot built by Html.
  • If one need to write multiple lines of code for simpler things, then it increases the complexity and takes more time.
  • Html can be used for first authentication step in the web but it's not robust. So, Security features are not good in Html and it provides only limited security.
  • Its error is costly because one small type of error could lead to non-functioning of webpages. there is no debugging tool is provided to detecterrors easily.
  • To better presentation of web pages along with Html, other languages like CSS And JavaScript needs to be learned.

HTML History

HTML was created by Tim Berners-Lee in 1991. The first ever version of HTML was HTML 1.0 but the first standard version was HTML 2.0 which was published in 1995.

  • 1991: Tim Berners-Lee invented HTML
  • 1992: Dave Raggett from Hewlett-Packard's Labs in Bristol, England who was one of the enthusiastic of HTML, met Tim and upon returning England, Dave wrote a richer version of HTML called HTML+.
  • November 24, 1995: HTML 2.0 was published as RFC 1866.
  • January 14, 1997: HTML 3.2 was published as a W3C Recommendation. It was the first version developed and standardized exclusively by the W3C, as the IETF had closed its HTML Working Group on September 12, 1996.
  • December 18, 1997: HTML 4.0 was published as a W3C Recommendation.
  • December 24, 1999: HTML 4.01 was published as a W3C Recommendation.
  • October 28, 2014: HTML5 was published as a W3C Recommendation.
  • November 1, 2016: HTML5.1 was published as a W3C Recommendation.
  • December 14, 2017: HTML5.2 was published as a W3C Recommendation.
