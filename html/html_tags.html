What is HTML Tag?

  • A tag is unit of markup. It is a set of symbols define in HTML to have spesial meaning.
  • The tag instructs the browser how to format and display the web page and is written directly into the text editor.
  • Tag start with opening angle bracket (<) followed by a keyword(Tag Name) and end with closing angle bracket (>). These symbols are known as angle brackets.
  • Every tag in HTML has a meaning and it is usually straight forward. For example, <p> is a tag that means to create paregraph of text/element appearing between it.
  • HTML tags normally comes in pair like <p> and </p>. They are usually paired to start and end the tag instuction.
  • There are two types of Tags:
    1. Start Tag.
    2. End Tag.

1. Start Tag

Start tag is the first HTML tag used to begain an effect. Start tag is known as opening tag. The start tag is surrounded by opening (<) and closing (>) angle brackets.

Syntax:

<tag name>

Example

<h1>This is a Heading.</h1>
<p>This is a Paragraph.</p> Output
RK Editor

Output :

This is a Heading.

This is a paragraph.

Example Explained:

The tags <h1> and <p> are the start tags. <h1> defines the taxt/element between it is heading. And <p> defines the text/element between it is Paragraph.

Tip : You can learn much more about Headings and Pragraph in our HTML Basic chapter. Also in our HTML Structure chapter you can learn about the <html> and <body> tags used in aboved example.

NOTE : There is always html and body tag in our examples. For convinence we don't write it again and again. Never miss this tags.

2. End Tag

End tag is the second HTML tag used to end an effect of open tag. End tag is known as closing tag. Name of end tag is same as that of start tag but the name of end tag is preceded by forward slah (/) inside of angle brackets, directly after the left angle bracket.

Syntax:

</tag name>

Example

<h1>This is a Heading.</h1>
<p>This is a Paragraph.</p> Output
RK Editor

Output :

This is a Heading.

This is a paragraph.

Example Explained:

The tags </h1> and </p> are the end tags. </h1> defines the end of heading. And </p> defines the end of Paragraph.

NOTE : Never Skip the End Tag. Errors may occur if you forget the end tag. but some times Some HTML elements will display correctly, even if you forget the end tag.

What is HTML Elements?

An element is a fundamental component of the structur of a text document. The combination of Start tag (<p>), End tag (</p>) and Content ("This is a Paragraph.") forms a HTML Element. The HTML element is everything from the start tag to the end tag. The elements are the bits that make up a webpage.

Content:

The text or other elements contained between the start and end tags of an HTML element is called content. Content is may be text or other element or combination of both.

Syntax:

<tag name>Some Content</tag name>

Examples:

Start tag Content End Tag
<h1> This is a Heading. </h1>
<p> This is a Paragraph. </p>
<br> - -

NOTE : Elements with no content and no end tag are known as empty elements.

HTML Tag vs. HTML Element

HTML element is the collection of start tag, an end tag and everything in between. On the other hand a HTML tag (either opening or closing) is used to mark the start or end of an element

Example

<p>This is Paragraph.</p>

Example Explained:

In above example <p> and </p> are tags. The whole example conaining tags and content is Element

Empty Elements

Elements with no content and no end tag are known as empty elements or empty tags or unclosed tags. Empty elements, which do not wrap around content will close themselves so they are also known as selfcloseing elements.

The line-break (<br>) :

A line break doesnâ€™t hold any content so line-break tag has no closing tag. line-break tag is use to break the line of code.

<br> tag forces a line break wherever you placed it. It tells the browser to wrep the text that follows onto a new line without inserting any extra space between the lnes. The <br> tag used when you want to end a line.

For an entire blank line write <br> tag on that line. Similarly for n number of blank lines type (n+1) times <br> tag. For Example: For 2 blank line write 3 <br> tags

Example

<p>This paragraph have <br> a line break.</p>
<p>This paragraph have <br> <br> <br> two blank line.</p> Output

This paragraph have
a line break.

This paragraph have


two blank line.

The Horizontal Rule (<hr>) :

<hr> tag is a horizontal rule and it is also known as horizontal line. This tag is used to put a line across the webpage. A web page can be devided into seperate sections by using horizontal rule. This tag is mostly used in decorative purposes.

Example

<h1>This is a Heading.</h1>
<hr>
<p>This is a Paragraph.</p> Output
RK Editor

Output :

This is a Heading.

This is a Paragraph.

Nesting HTML Elements

You can put elements inside other elements too this is called nesting. HTML Elements can be nested within each other. All HTML documents consist of nested HTML elements.

Following example consist of four HTML Elements(<html>, <body>, <b>, <i>)

Example

<html>
<body>
<b>
<i>This text is in both bold and italic formates.</i>
</b>
</body>
</html>
 Output
RK Editor

Output :

This text is in both bold and italic formates.

Example Explained:

HTML file always start with <html> tag and end with </html>. The <html> tag represents root of an HTML document hence also called as root element. It is a container of all elements (except <!Doctype>).

Then, inside the <html> element there is a <body> element.
<body>
<b>
<i>This text is in both bold and italic formates.</i>
</b>
</body>

The actual content of the web page that will be displayed on browser will appear in body section of document. The body section start with <body> tag and end with </body> tag.

Then, inside the <body> element there is a <b> element.
<b>
<i>This text is in both bold and italic formates.</i>
</b>

The <b> is bold tag. The taxt appearing between start tag (<b>) and end tag (</b>) will be displayed in bold letters.

Then, inside the <b> element there is a <i> element.
<i>This text is in both bold and italic formates.</i>

The <i> is an italic tag. The taxt appearing between start tag (<i>) and end tag (</i>) will be displayed in italics.

We can see that HTML Elements can be nested with in each other. In above Example we can observe that the text is displyed in both bold and italic formates.

Also we can observe that italic element is completely within the bold element. The elements have to open and close correctly so that they are clearly inside or outside one another.

NOTE : The HTML Elements Should be nested in proper sequence. Not like this - <i><b></i></b>.

Never skip the End Tag

Never Skip the End Tag. Errors may occur if you forget the end tag. It is not pssible to say that what error will come when we forget the end tag. but some HTML elements like paragraph (<p>) will display correctly, even if you forget the end tag.

Example

<p>This is a Paragraph.</p>
<p>This is a Paragraph. Output
RK Editor

Output :

This is a Paragraph.

This is a Paragraph.

Case-sensitivity of HTML Tags

HTML tags are not case sensitive<h1>means the same as<H1>.

But in XHTML they are case-sensitive and the tag <h1> is different from the tag <H1>.

Example

<h1>This is a Heading.</h1>
<H1>This is a Heading.</H1> Output
RK Editor

Output :

This is a Heading.

This is a Heading.

Tag Description
<!Doctype> specifies the type of document.

<body> specifies the body section of HTML document.
<p> specifies a Paragraph.
<h1>-<h6> specifies headings from level 1 to level 6.
<br> specifies a line break.
<hr> specifies thematic break between elements (specifies a horizontal rule/line).
<b> specifies bold text.
<i> specifies italic text.
