What is HTML Attribute?

  • All elements can have attributes. And every attributes containes additional pieces of information about tags.
  • We can also say that attributes are the spesial words that can modifie the elements.
  • An attribute used to define the characteristics of an HTML element.
  • Attribute is specified in start tag of HTML Element.
  • All attributes are made up of two parts −
    1. Attribute Name
    2. Attribute Value

Syntax:

<tag name attribute_name="attribute_value"> Some Content <tag name>

Example

<h1 style="color:blueviolet"> This is heading with blueviolet text color. </h1>
