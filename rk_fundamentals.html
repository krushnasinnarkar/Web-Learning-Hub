Number Systems

Every letter, symbol etc. that we write to computer, it gets converted into machine language. This machine language comprises of numbers. In order to understand the language used by digital systems it is crucial to have a better understanding of number system.

Binary N. S.

The binary number system is base-2 number system. Binary Number System consists of two digits only i.e. 0 and 1. Thus, the base of the binary number system is 2 as the available digits in this number system is 2. The other numbers can be expressed with these two digits.

Read More...
Octal N. S.

The Octal number system is base-8 number system. It consists of 8 digits i.e. from 0 to 7. Thus, the base of the Octal Number System is 8 as the available digits in this number system is 8. It is much easier to handle array of octal numbers in comparison to binary numbers.

Read More...
Decimal N. S.

The Decimal Number System is base-10 number system. It consists of 10 digits i.e. from 0 to 10. Thus, the base of the Decimal N.S. is 10 as the available digits in this number system is 10. Using these 10 digits we can define any number in this system no matter how large it is.

Read More...
Hexadecimal

The Hexadecimal N.S. is base-16 number system. It consists of 10 digits and 6 alphabets.The 10 digits from 0 to 9 and the alphabets A, B, C, D, E and F represents 10, 11, 12, 13, 14 and 15 respectively. Thus, It is mostly used in case of microprocessors and microcontrollers.
